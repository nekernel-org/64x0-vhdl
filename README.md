# Open64x0 Core.

## Brief:

Here you will find samples, documentation and the VHDL sources.
The ROM needs the 64asm from NeKernel's `toolchain`, please get that as well.

## Cloning:

git clone git@github.com:nekernel-org/64x0-vhdl.git
